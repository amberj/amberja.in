Go back to "The Matrix in HTML" implementation.

This is implementation of something that looks like matrix from The Matrix series/trilogy (or atleast it appears so). I did this stupid stuff one night when I was bored. Do you like it?

This matrix implementation uses the concept of HTML Meta Refresh. The Wikipedia article clealy says: "This method is deprecated and should not be used". W3Schools says: "Note: The attribute value "Refresh" is nonstandard, and should not be used. "Refresh" takes the control of a page away from the user. Using "Refresh" will cause a failure in W3C's Web Content Accessibility Guidelines".

NOTES:
  • On very old and slow dialup connections, there may be some noticeble time required to load the pages initially but once the pages are saved in rowsing history of web browser, everything runs fine.
  • The HTML title tag should display "The Matrix" but when the page is redirecting it displays "Loading" in firefox (and maybe other browsers too). I don't think that there is any workaround for this.
JWZ's website has a similar but better alternative solution to this.
